# Real-Time Chat Application

A full-stack real-time chat application built using React, Node.js, Express.js, and Socket.IO. Users can send and receive messages instantly through real-time WebSocket communication.

## Live Demo

https://renewed-delight-production-28c0.up.railway.app

## Features

- Real-time messaging
- Instant message delivery using Socket.IO
- Message timestamps
- Chat history during server session
- Dark and Light mode
- Clear chat option
- Auto scroll to latest message
- Responsive chat interface

## Tech Stack

### Frontend
- React.js
- Vite
- Socket.IO Client
- CSS

### Backend
- Node.js
- Express.js
- Socket.IO
- CORS

### Deployment
- Railway

## How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/ranjansah809/real-time-chat-app.git
cd real-time-chat-app
```

### 2. Start the Backend

```bash
cd backend
npm install
npm start
```

### 3. Start the Frontend

Open another terminal:

```bash
cd fronted
npm install
npm run dev
```

## Real-Time Testing

Open the application in two different browser windows or an Incognito window. Send a message from one window and it will instantly appear in the other window.
