# 💬 Real-Time Chat App

A responsive chat application built using React and Vite. The project provides a clean and interactive chat interface with dark mode support and responsive design.

## 🚀 Live Demo

🔗 https://real-time-chat-app-neon-two.vercel.app/

## ✨ Features

- 💬 Interactive chat interface
- 🌙 Dark mode support
- 🧹 Clear chat functionality
- 📱 Responsive user interface
- ⚡ Fast performance with Vite
- 🎨 Clean and modern design

## 🛠️ Tech Stack

- React.js
- JavaScript
- Vite
- HTML5
- CSS3

## 📂 Project Structure

- `frontend/` - React frontend application
- `src/` - Application source code
- `App.jsx` - Main React component
- `App.css` - Application styling

## ⚙️ Installation

Clone the repository:

git clone https://github.com/ranjansah809/real-time-chat-app.git

Navigate to the frontend directory:

cd real-time-chat-app/frontend

Install dependencies:

npm install

Run the development server:

npm run dev
